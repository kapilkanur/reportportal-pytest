# reportportal-pytest
### Overview

This repository is a **simple sample project demonstrating Pytest integration with ReportPortal**. It shows how automated test execution results can be sent to ReportPortal for centralized reporting, analysis, and visualization.

The project uses a minimal Pytest setup with basic test cases and configuration to help users quickly understand how ReportPortal captures test launches, test names, statuses, and logs. It is intended as a lightweight reference or starter template for teams integrating ReportPortal into Python-based test automation frameworks.
